## ‚ú® Features

* **Full CRUD Functionality:** Create, read, update, and delete tasks.
* **User Authentication:** Secure login and registration using Spring Security (JWT).
* **Organization:** Assign tasks to categories/projects, set priorities, and add due dates.
* **Dashboard:** A quick overview of pending, completed, and upcoming tasks.
* **Modern Desktop UI:** A responsive and clean user interface built with JavaFX and FXML.

## üõ†Ô∏è Tech Stack

This project demonstrates proficiency across the modern Java ecosystem:

### Backend (Spring Boot)
* **Framework:** Spring Boot 3
* **Security:** Spring Security (with JWT for token-based authentication)
* **Database:** Spring Data JPA (Hibernate)
* **Validation:** Spring Validation for API endpoints
* **Database:** PostgreSQL (or H2 for testing)
* **Build:** Maven (or Gradle)

### Frontend (JavaFX)
* **Framework:** JavaFX 21
* **UI Structure:** FXML
* **Styling:** CSS
* **API Consumption:** Java 11+ `HttpClient` (or Retrofit) for communicating with the REST API.
